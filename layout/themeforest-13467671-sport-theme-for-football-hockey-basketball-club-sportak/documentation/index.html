Installation

  1. Theme use big amount of configuration settings which can be installed via demo content import.
  2. Configuration settings use post ID for establish links so import demo content will work correctly only on fresh WP installation.
  3. Download the Theme archive and extract the files it contains.
  4. Using an FTP client to access your host web server, create sport-ak directory to save Theme in the wp-content/themes directory provided by WordPress. This theme provide this directory as part of the archive.
  5. Upload the Theme files to the new directory on your host server.
  6. Activate theme.

Import data with both checkboxes enabled

After import you need:

  1. Enable Visual Composer for Page and azsc_match.
  2. Select "Secondary" menu in "WP Custom Menu" element of "Footer" VC Widget.
  3. Install Revolution slider plugin.
  4. Create revolution slider and choose it in "Slider" VC Widget.
  5. Install and configure Latest Tweets plugin.

Import configuration

After import you need:

  1. Select "Secondary" menu in "WP Custom Menu" element of "Footer" VC Widget.
  2. Create contact forms by use templates in "contact-form-7" folder of this documentation
  3. Select appropriate contact form in "Contact Form 7" element settings of VC Widgets and Pages
  4. Place VC Widgets in sidebars and configure visibility rules
  5. Use AZEXO elements in Visual Composer to make pages

Code navigation

  1. list.php - universal content template
  2. *.php in theme root directory - standard WordPres template files
  3. yarpp-template-default.php - YARPP plugin related posts template
  4. azexo/exporter/exporter.php - export theme settings for developers
  5. azexo/importer/*.php - import demo content code
  6. azexo/icons/*.php - Visual Composer additional icons support
  7. azexo/importer/data - demo content files
  8. azexo/options/*.json - default theme options for skin
  9. azexo/less-variables.php - link less variables with theme settings
  10. azexo/options-init.php - REDUX framework theme settings configuration
  11. azexo/tgm-init.php - TGM class configuration
  12. azexo/vc_extend.php - Visual Composer extend configuration
  13. azexo/woocommerce.php - WooCommerce related theme part
  14. less/*.less - theme styles (table of contents in theme style.css file)
  15. fields - fields templates used in theme
  16. css - fonts and animation CSS libraries
  17. fonts - fonts files
  18. images - images used by theme
  19. js/*.js - JS libraries
  20. js/azexo.js - main JS theme script
  21. lang - theme translation files
  22. page-templates/*.php - WordPress page templates
  23. plugins - WordPress plugins distributives required by theme
  24. post-like-system - post like feature files
  25. redux-framework - REDUX theme settings framework
  26. vendor_support - REDUX vendor support extension
  27. template-parts - template parts which reuse in other theme templates
  28. tgm - TGM class
  29. vc_templates - new and overridden Visual Composer elements
  30. widgets - widgets provided by theme
  31. wp-less - wp-less plugin with php less compiler
